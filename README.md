### Back-end:
* Node.js
* PostgreSQL database (Supabase)
* RESTful API ([API Reference](https://www.nps.gov/subjects/developer/api-documentation.htm))
* Express package
* ES6 Templating
* Form validation
* Full C.R.U.D. functionality
* Bcrypt user authentication & validation
